# Sign Language Recognizer (SLR) – Final Year Project

This repository contains the code and resources for a Sign Language Recognizer (SLR), developed as a Final Year Project. The goal of this project is to leverage computer vision and machine learning to translate sign language gestures into text, thereby bridging the communication gap for the deaf and hard-of-hearing community.

## About

Sign Language Recognizer (SLR) is designed to recognize and interpret sign language gestures using modern computer vision techniques. This system aims to aid communication accessibility and can serve as a foundation for more advanced real-time translation tools.

---

## Features

- Real-time or batch processing of sign language gestures.
- Converts gestures to readable text.
- Can be extended to support more signs and languages.
- Modular, extensible codebase for experimentation.

---

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ADVAIT135/slr-fyp.git
   cd slr-fyp
   ```

2. **Install dependencies:**
   - Ensure you have Python 3.x and pip installed.
   - (If requirements.txt is present, run)
     ```bash
     pip install -r requirements.txt
     ```
   - Otherwise, install dependencies as specified in the documentation or source code.

---

## Usage

- Check the main script or notebook for instructions on running the recognizer.
- Input sign language videos or use a webcam for real-time recognition (if supported).
- Results will be displayed as corresponding text output.
